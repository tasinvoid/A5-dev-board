Question-1: What are the different ways to select an element in the DOM?

Methods to select elements from the DOM are - getElementById querySelector querySelectorAll getElementsByClassName getElementByTagName.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.innerHTML- Shows the full html inside the tag 2.innerText- shows just the text inside the tag 3.textContent- Shows plain text without any formatting including all text within the element and its child

Question-3: What is event delegation in the DOM?

Using a single event listener to a parent element instead of each child elements

Question-4: What is event bubbling in the DOM?

Event starts at the target element and then bubble up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

1.To create element const newElement = document.createElement('div'); newElement.textContent = 'Hello, World!'; 2.To add element const parentElement = document.getElementById('parent'); parentElement.appendChild(newElement); 3.Remove an element using remove() const elementToRemove = document.getElementById('child'); elementToRemove.remove();
